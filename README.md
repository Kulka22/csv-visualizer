# csv-visualizer

Тестовое задание на позицию **Стажёр C# / .NET разработчик**.

Frontend был предоставлен в готовом виде.  
Задача заключалась в реализации backend-части на **ASP.NET Core**, чтобы приложение корректно работало.

---

## Описание

Приложение позволяет:

- загрузить CSV-файл с данными
- определить категориальные и числовые поля
- построить столбчатый график на основе выбранных параметров

Backend реализует:
- загрузку и обработку CSV-файлов
- in-memory хранение данных
- агрегацию (SUM / AVG / COUNT)
- фильтрацию категорий
- сортировку и Top N
- условное форматирование
- расчёт и коррекцию опорной линии

Взаимодействие с frontend выполнено строго по контракту, описанному в задании и самом frontend'е.

---

## Стек

- **ASP.NET Core**
- **CsvHelper** (для чтения CSV)
- **System.Text.Json**
- In-memory хранилище данных
- Vanilla JavaScript (frontend — предоставлен)
